# shop
redux-toolkit nestjs typeorm

## how to use
npm install

front: npm run dev

back: npm run start:dev


## required

.env

PORT=3065

COOKIE_SECRET=

DB_USERNAME=root

DB_PASSWORD=

DB_DATABASE=shop
